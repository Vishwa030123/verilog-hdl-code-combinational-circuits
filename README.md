# verilog-hdl-code-combinational-circuits

## Overview
Implementation of combinational logic circuits in Verilog HDL demonstrating different abstraction levels: gate-level, dataflow, and behavioral modeling. Functional verification and simulation have been carried out using EDA Playground with Synopsys VCS

## Features
- Synthesizable verilog HDL code
- Includes major Abstraction level code including RTL level.

## Tech Stack
- Verilog
- EDA playground Synopsys VCS

## Results / Output
-Functional verification performed and simulation performed in EDA tool.
